Welcome to user API!

This is an API for test only and it should be that way!


There are several endpoints can be used as follow:

  1. GET /user to fetch all list of users
  2. GET /user/{id} to fetch a user with id {id}
  3. POST /user to add a new user
    Body used: name (String), age (Int), salary (Double), isMarried (Boolean)
  4. PUT /user/{id} to update a user with id {id}
